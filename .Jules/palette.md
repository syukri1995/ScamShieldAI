## 2024-05-24 - Enhancing Streamlit Input Accessibility and Tooltips
**Learning:** Streamlit's abstraction makes direct DOM manipulation (like adding ARIA labels to standard inputs) difficult. However, native Streamlit parameters like `help=` provide built-in tooltips that improve both UX and accessibility context without needing hacky HTML injections. For custom injected HTML components, standard ARIA attributes (`role="listbox"`, `role="option"`) must be manually assigned via JS `setAttribute` to ensure screen readers can parse dynamic overlays correctly.
**Action:** Use native Streamlit parameters (`help`) for standard components, and explicitly set ARIA attributes via JS when injecting custom HTML overlays.
